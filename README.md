# MapShare
An easy location sharing app

# Setup

## Required Aptitude Packages

- python3-dev
- python3.11-venv
- libmysqlclient-dev
- python3-dotenv

## Python3 Virtual Environment

Create virtualenv using command

`python3 -m venv venv`

## Environment Variables

Copy .env.example to create .env file

`cp .env.example .env`
