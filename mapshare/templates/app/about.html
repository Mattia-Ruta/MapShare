About Mapshare

What is Mapshare?

Mapshare is a web application that allows you to easily share and find locations using a revolutionary system called Mapcodes. My goal is to make location sharing easier and accessible to everyone.

What are Mapcodes?

Mapcodes are short, easy-to-remember codes that represent specific locations on Earth. Think of them as a more user-friendly version of latitude and longitude coordinates. Some key features of mapcodes:

  • Typically 4-7 characters long
  • They can represent any location on Earth with high precision (around 5m²)
  • Easy to read, say, and remember (think of it like a UK postcode but for the whole world)
  • They Work internationally and are language-independent. Mapcodes are the same no matter where it is in the world!

For example, instead of saying "51.5007° N, 0.1246° W" for Big Ben in London, you could just use the Mapcode "JJ.66". Isn't that a lot easier?

Territories and Mapcodes

Mapcodes can be used with or without territory codes. When a territory code is used, it allows for shorter mapcodes within that specific region. For example:

  • The international mapcode for Big Ben is "GCPVJ.QGQV"
  • But within the GBR territory, it can simply be "JJ.66"

This system allows for easier local use, but be careful when sharing across different territories! Mapshare automatically provides the most relevant mapcode based on the location, but also offers alternative options when available.

More info about mapcodes here on the official site.

How Mapshare Works

  1. Find a Location: Use our map interface to pinpoint any location in the world.
  2. Get the Mapcode: Once you've selected a location, Mapshare will generate the corresponding Mapcode.
  3. Share: Copy the Mapcode and share it with others via any messaging app.
  4. Locate: When someone receives a Mapcode, they can enter it on Mapshare to see the exact location on the map.

Why Use Mapshare?

  • Simplicity: Mapcodes are much easier to communicate than long GPS coordinates.
  • Accuracy: Pinpoint specific locations, even in areas without formal addressing systems.
  • Universal: Works anywhere in the world, making it great for international use.
  • Privacy: Mapcodes don't contain any inherent information about the location, enhancing privacy.
